Windows 11 Setup Tips
  • Disable Auto HDR: Prevent double-tone-mapping by turning this off.
    Settings > System > Display > HDR > Auto HDR [OFF]
  • Disable ACM: Ensure "Automatic Color Management" is off to avoid conflict.
    Settings > System > Display > Advanced Display > Automatically manage color [OFF]
  • Windows SDR Slider: This slider does NOT affect RTX HDR brightness. It only affects the desktop/UI background.
  • Apply In-Game: Use the overlay while the game is running to apply these values.
    Press Alt+Z > Game Filter > RTX HDR
